安全意识培训考试题目样题
发表于 2021-05-19| 更新于 2021-05-19

样题

单项选择题：（每题3分，共20题60分）

1、自2013年以来已有近（ ）条数据泄露，平均每秒有74条数据泄露。

A 200万

B 800万

C 1000万

D 130亿

2、（ ）于 1995 年 10 月 24 日通过《关于在个人数据处理过程中保护当事人及此类数据自由流通的指令》，简称《个人数据保护指令》，这一指令要求各国采取统一标准对个人数据进行保护。

A 欧洲议会和欧盟理事会

B 北约以及联盟国

C 印度以及联盟国

D 大不列颠条约

3、 2016年4月14日，欧洲议会投票通过了商讨四年的（ ），该条例将在2018年5月25日正式生效。新条例的通过意味着欧盟对个人信息保护及其监管达到了前所未有的高度，堪称史上最严格的数据保护条例。

A 《一般数据保护条例》

B 《欧盟数据安全条约》

C 《北约数据安全条约》

D 《美欧数据安全条约》

4、 《网络安全法》涉及数据安全条款共11条，从（ ）安全三个层面构建起我国数据安全整体保障体系框架。

A 运维自身安全、公民信息安全、机房重要数据资源

B 数据库自身安全、APP信息安全、战略数据资源

C 网站自身安全、企业信息安全、公民重要数据资源

D 数据自身安全、公民个人信息安全、国家重要数据资源

5、违反国家规定，对计算机信息系统功能进行（ ），造成计算机信息系统不能正常运行，后果严重的，处五年以下有期徒刑或者拘役；后果特别严重的，处五年以上有期徒刑。

A 删除、修改、增加、干扰

B 破坏、处理、丢失、放弃

C 泄漏、掩埋、备份、异地

D 拷贝、粘贴、复制、剪切

6、网络运营者以经营为目的收集重要数据或个人敏感信息的，应向所在地（ ）。备案内容包括收集使用规则，收集使用的目的、规模、方式、范围、类型、期限等，不包括数据内容本身。

A 网信部门备案

B 计算机维修中心

C 公安部监督中心

D IDC机房中心

7、“公民个人信息，是指以电子或者其他方式记录的能够单独或者与其他信息结合识别公民（ ）的各种信息，包括但不限于公民的姓名、出生日期、身份证件号码、个人生物识别信息、住址、电话号码等。”

A 公司身份

B 个人身份

C 家庭身份

D 群体身份

8、（ ）是指一旦泄露、非法提供或滥用可能危害人身和财产安全，极易导致个人名誉、身心健康受到损害或歧视性待遇等的个人信息。

A 公司敏感信息

B 个人敏感信息

C 文件敏感信息

D 家庭敏感信息

9、个人信息控制者宜对需展示的个人信息采取（ ）等措施，降低个人信息在展示环节的泄露风险。

A 标识化处理

B 简单化处理

C 复杂化处理

D 明文化处理

10、2017年5月12日，全球范围爆发针对Windows操作系统的勒索软件( )感染事件。该病毒利用此前美国国家安全局网络武器库泄露的WindowsSMB服务漏洞进行攻击。

A CIH

B WannaCry

C Melissa

D Sasser

11、 2018年8月20日，公安机关成功侦破一起特大规模的数据窃取案，公司几名主要负责人涉嫌非法窃取用户信息30亿条。三家涉案公司则通过与运营商合作，通过（ ）等各种数据，瑞智华胜则用来流量数据处理、推广获利变现。

A 用户账户，爆破密码

B 控制域名，获取后台

C 劫持链接，控制DNS

D 劫持流量，清洗用户cookie

12、 后台账户的管理和使用十分重要，（ ）行为是正确操作账号使用的方式。

A 借用账号给测试部门使用

B 忽视日常审计、账号共享

C 账号放在Github等公网上

D 一人一号限期使用，到期进行账号密码收回

13、在日常开发测试中、经常会添加一些比较重要的数据以方便项目正常使用进行，（ ）行为会导致信息泄漏。

A 用户数据内网的网站数据库中

B 账号密码放在公网可访问的Web页面里

C 个人信息进行加密且不可逆的算法

D 开发人员电脑连接不明热点网络

14、在日常开发测试中、经常会添加一些比较重要的数据以方便项目正常使用进行，（ ）行为会导致信息泄漏。

A 用户数据内网的网站数据库中

B 账号密码放在公网可访问的Web页面里

C 个人信息进行加密且不可逆的算法

D 开发人员电脑连接安全网络

15、网络运营者应当对其收集的用户信息（ ），并建立健全用户信息保护制度。

A 简单保存

B 严格保密

C 明文存储

D 不做加密

16、网络运营者应当对其收集的用户信息（ ），并建立健全用户信息保护制度。

A 简单保存

B 严格保密

C 明文存储

D 不做加密

17、从事个人信息处理岗位上的相关人员签署（ ），对大量接触个人敏感信息的人员进行背景审查。

A 工作协议

B 人身安全

C 保密协议

D 医保协议

18、个人信息处理岗位上的相关人员在调离岗位或终止劳动合同时，继续履行（ ）。

A 沟通义务

B 传播义务

C 保密义务

D 访问义务

19、对外（ ）披露个人信息，应向个人信息主体提供撤回授权同意的方法。

A 共享、转让、公开

B 拷贝、复制、粘贴

C 保存、掩埋、管理

D 披露、传播、删除

20、注销账户后，应及时（）其个人信息或做匿名化处理。

A 保留

B 拷贝

C 传播

D 删除

简答题（每题10分，共4题40分）

1、个人信息的使用乱象丛生，个人信息保护成了十分重要的突出问题，请列举出个人信息被非法利用所导致的一些后果。

2、刑法第286条，“造成计算机信息系统不能正常运行，后果严重的，处五年以下有期徒刑或者拘役；后果特别严重的，处五年以上有期徒刑”，触犯这种严重后果的行为有哪几种？

3、全球范围爆发勒索病毒，用户需支付比特币才能取回文件，否则赎金翻倍或是文件被彻底删除。全球100多个国家数十万用户中招，国内的哪些行业遭受不同程度的影响。

4、刑法第二百五十三条之一规定的“公民个人信息”，是指以电子或者其他方式记录的能够单独或者与其他信息结合识别特定自然人身份，或者反映特定自然人活动情况的各种信息。请列举出属于公民的个人隐私信息的特征。（回答4个及以上）
