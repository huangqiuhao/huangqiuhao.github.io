移动应用安全| iOS取证/调试/静态分析/动态调试/逆向工程/工具整理
发表于 2019-08-28| 更新于 2019-09-24|Android APP安全
1、iOS调试工具
  • 类转储Z - 提取Objective-C版本2 ABI的类接口。
  • IDA Pro - IDA是Windows，Linux或Mac OS X托管的多处理器反汇编程序和调试程序，它提供了许多功能，很难描述它们。
  • HopperApp - Hopper是OS X和Linux的逆向工程工具，可让您对32/64位Intel Mac，Linux，Windows和iOS可执行文件进行反汇编，反编译和调试！
  • otool - otool命令显示目标文件或库的指定部分。它也可以拆卸
  • strings - strings是一个简单的实用程序，它将打印给定二进制文件中的所有字符串。
  • nm - nm是一个显示给定二进制符号表的实用程序。
2、iOS取证工具
  • iPhone分析器 - iPhone Analzyer允许您从iOS设备中进行法医检查或恢复日期。
  • iPhone备份分析器
  • iPhone备份浏览器
  • Keychain Dumper - 一种在iOS设备被越狱后检查攻击者可以使用哪些钥匙串项目的工具。
  • Mac-robber - mac-robber是一种数字调查工具，可从已安装文件系统中的已分配文件中收集数据。在分析实时系统或分析实验室中的死系统时，这在事件响应期间非常有用。
  • 移动同步浏览器 - 访问您的iOS备份
  • Mobilyze - Mobilyze是一种移动数据分类工具，旨在让用户立即访问iOS和Android设备上的数据。
  • 氧气取证套件
  • Paraben设备癫痫发作
  • SleuthKit - SleuthKit®是一系列命令行工具和C库，允许您分析磁盘映像并从中恢复文件。它在Autopsy和许多其他开源和商业取证工具的幕后使用。
3、iOS动态分析工具
  • Appmon - AppMon是一个自动化框架，用于监控和篡改iOS，Mac OS X和Android应用程序（即将推出）上的本机应用程序的系统API调用。您可以将其称为本机移动应用程序的GreaseMonkey。
  • cycript - Cycript允许开发人员使用Objective-C ++和JavaScript语法的混合，通过具有语法突出显示和制表符完成功能的交互式控制台，在iOS或Mac OS X上探索和修改正在运行的应用程序。
  • Clutch - Clutch是一款高速iOS解密工具。Clutch支持iPhone，iPod Touch和iPad以及所有iOS版本，架构类型和大多数二进制文件。离合器仅用于教育目的和安全研究。
  • DIOS - DiOS是一个实用的系统，可以对iOS应用程序进行自动动态隐私分析。DiOS提供高度可扩展且完全自动化的解决方案，用于安排从官方Apple App Store中的应用程序，以便对iOS设备进行隐私分析。在自动执行应用程序的同时，使用随机和智能执行策略模拟用户交互，并跟踪敏感API调用以及网络连接。
  • iNalyzer - AppSec Labs iNalyzer是一个用于操纵iOS应用程序，篡改参数和方法的框架; 无需任何来源！AppSec实验室iNalyzer针对封闭式应用程序，将痛苦的黑盒子变成了自动灰盒子。
  • IDB -美洲开发银行是简化的iOS和渗透测试研究的一些常见任务的工具
  • iOS SSL Kill Switch - Blackbox工具，可在iOS Apps中禁用SSL证书验证 - 包括证书固定
  • iOS SSL Kill Switch2 - Blackbox工具，可在iOS和OS X应用程序中禁用SSL证书验证 - 包括证书固定
  • Introspy-iOS - Blackbox工具，有助于了解iOS应用程序在运行时的功能，并帮助识别潜在的安全问题。
  • needle - 一个开源的模块化框架，用于简化对iOS应用程序进行安全评估的过程。
  • snoop-it - 一种协助iOS应用程序进行安全评估和动态分析的工具
  • XSecurity - Xcode中的安全插件，以及用于iOS应用程序开发的Clang Static Analyzer Checkers。此插件旨在通过在创建漏洞时检测漏洞来减少开发期间的漏洞。
4、iOS逆向工程工具
  • Flex-2 - Flex - 2使您能够修改应用程序并改变其行为，而无需了解iOS开发的复杂性。
  • iSpy - iOS的逆向工程框架
  • iRET - iOS逆向工程工具包是一个工具包，旨在自动执行与iOS渗透测试相关的许多常见任务。
  • cycript - Cycript允许开发人员使用Objective-C ++和JavaScript语法的混合，通过具有语法突出显示和制表符完成功能的交互式控制台，在iOS或Mac OS X上探索和修改正在运行的应用程序。
  • Logify - Logify采用包含类接口的Objective-C头文件，并生成一个挂钩给定类中所有方法的Logos文件，并为每个挂钩记录方法调用（带参数）到syslog
