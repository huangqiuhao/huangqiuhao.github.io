老司机福利：构建私人网盘，远离数据泄露。
发表于 2019-08-14| 更新于 2019-08-22|私人网盘
简介

网盘，一度成为人们不可或缺的一部分，几年前网盘争夺战到处弥漫，1T、10T、30T存储空间无处不在，现如今要么关闭，要么收费，随着国内网盘厂商纷纷的关闭，目前也只有百度云、微云还可以使用，但是云的不安全问题今年也变得非常多，数据大爆炸的时代，需要一个安全，稳定，可控的网盘存储，毕竟东西放在别人那里，说不上什么安全。Seafile 是一款开源的企业云盘，注重可靠性和性能。支持 Windows, Mac, Linux, iOS, Android 平台。支持文件同步或者直接挂载到本地访问。我们今天就来搭建属于自己的网盘系统，把最高权限留给自己。

（以下步骤均在Center os 7.0上操作，命令与7.0以下版本有所不同）

1.安装Python和Mysql环境

2.启动服务

3.配置数据库信息

输入：/usr/bin/mysql_secure_installation 回车两下，回到达New password，输入密码，回车再次重复密码，其余的一路回车，配置完成。

4.下载Seafile安装包，解压

wget https://mc.qcloudimg.com/static/archive/3d8addbe52be88df4f6139ec7e35b453/seafile-server_5.1.4_x86-64.tar.gz

如果提示wget不存在，使用yum install wget安装wget组件即可。

解压：tar zxvf +文件

5.安装Seafile

字段说明：

[server name]随意

[ This server’s ip or domain ] 域名或者IP

其余的默认回车即可

[ 1 or 2 ] 字段选择 1

[ root password ] 填写第三步你配置的mysql 的密码

其余的回车即可，当然你也可以更改啦。

6.安装完成

7.启动Seafile

配置登陆邮箱和密码，可随意配置。

8.通过IP+端口号登录管理界面

暂时不绑定域名，需要域名的时候绑定即可

9.总结

恭喜！你拥有了属于自己的云盘！（若开启iptables需要把8082/tcp、8000/tcp端口加入）还是要告诫大家，数据需要多地备份，如果本地有大容量硬盘，还是存在本地，网络危害无处不在，别把重要的个人信息和账号密码存在云盘里，因为你不知道网络中有谁在盯着你！
